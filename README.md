## ❓ Common Questions

### How do I close tinyclip?

Right-click the tinyclip icon in the system tray and select “Exit” or “Close.”

### Does tinyclip start automatically when I turn on my computer?

By default, tinyclip sets itself to start on boot. You can change this in the settings menu inside the app.

### Is my clipboard data safe?

tinyclip stores clipboard items only on your computer. It does not send or share this data over the internet.

### Can tinyclip paste images?

tinyclip supports basic image copying and pasting. Complex formats may not always work.
